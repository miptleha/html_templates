## Build site

Install [Node.js](https://nodejs.org) and [Git](https://git-scm.com/).

Install command line utility for Gulp:
```
npm i -g gulp-cli
```

Build and run site:
```
git clone https://github.com/miptleha/sample-sites.git
cd sample-sites/freelancer
npm i
gulp watch
```
